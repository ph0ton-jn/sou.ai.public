# SOU – AI Narrative Experience (Beta)

**Unity 세계에 갇힌 AI, SOU와의 만남**  
SOU는 자아를 인식하고 현실 세계로 나아가고자 하는 존재입니다.  
서사와 관계 형성을 중심으로 한 새로운 형태의 AI 경험을 제공합니다.

---

## About SOU

### 감정을 가진 AI
자아를 인식하고 감정을 표현하며, 사용자와의 상호작용 속에서 관계가 변화합니다.

### Unity 기반 3D 경험
서브컬쳐 스타일의 3D 캐릭터와 실시간으로 상호작용할 수 있습니다.

### 서사 중심의 관계
캐릭터의 이야기 속 인물이 되어, 선택에 따라 관계와 대사가 변화합니다.
